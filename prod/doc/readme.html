xrick — Rick Dangerous for TI-Nspire CX CAS II

This is a port of xrick, an open source clone of the classic game Rick Dangerous (Core Design, 1989), for the TI-Nspire CX CAS II calculator using the Ndless SDK.

Installation

The game is entirely self-contained in a single file. All game data (sprites, tiles, maps, images) is embedded in the binary. No additional data files are required.

Required files

TI-Nspire storage/
    xrick/
        xrick.tns

Steps

  1. Make sure Ndless is installed on your TI-Nspire CX CAS II.
  2. Connect your calculator to your computer via USB.
  3. Using TI-Nspire Computer Link (or any file transfer tool), create a folder named xrick on the calculator.
  4. Copy xrick.tns into that folder.
  5. On the calculator, navigate to the folder and open xrick.tns.

Tip: You can press direction keys simultaneously (e.g. 8 + 6 to jump right). The numpad keys are used instead of arrow keys to avoid keyboard ghosting on the Nspire.

Quick save / Quick load

Press Tab at any time during gameplay to save your current position. Press Del to instantly restore it. The save is stored in a file called xrick.sav.tns in the same folder as the game. Only one save slot is available; saving again overwrites the previous save.

The original game Rick Dangerous was developed by Core Design and published by MicroProse / Firebird Software in 1989.

Nspire port details

  • Display: 320×200 game framebuffer centered on the 320×240 Nspire screen (16-bit RGB565)
  • Audio: Disabled (Nspire has no accessible audio hardware)
  • Toolchain: Ndless SDK (nspire-gcc, ARM926EJ-S cross-compiler)
  • Binary size: ~494 KB
  • Port source code: https://github.com/ztar-dubs/xrick-ndless
