xrick — Rick Dangerous pour TI-Nspire CX CAS II

Ceci est un portage de xrick, un clone open source du jeu classique Rick Dangerous (Core Design, 1989), pour la calculatrice TI-Nspire CX CAS II via le SDK Ndless.

Installation

Le jeu est entièrement autonome dans un seul fichier. Toutes les données du jeu (sprites, tiles, cartes, images) sont embarquées dans le binaire. Aucun fichier supplémentaire n'est nécessaire.

Fichiers requis

Stockage TI-Nspire/
    xrick/
        xrick.tns

Étapes

  1. Vérifiez que Ndless est installé sur votre TI-Nspire CX CAS II.
  2. Connectez votre calculatrice à votre ordinateur par USB.
  3. À l'aide du logiciel TI-Nspire Computer Link (ou tout autre outil de transfert), créez un dossier xrick sur la calculatrice.
  4. Copiez xrick.tns dans ce dossier.
  5. Sur la calculatrice, naviguez vers le dossier et ouvrez xrick.tns.

Enter / Ctrl / TabTirer / Utiliser l'arme

EFin de partie
EscQuitter

Astuce : Vous pouvez appuyer sur plusieurs touches de direction en même temps (par exemple 8 + 6 pour sauter vers la droite). Les touches du pavé numérique sont utilisées à la place des flèches pour éviter le ghosting clavier de la Nspire.

Détails du portage Nspire

  • Affichage : framebuffer 320×200 centré sur l'écran 320×240 de la Nspire (16 bits RGB565)
  • Audio : Désactivé (la Nspire n'a pas de matériel audio accessible)
  • Chaîne de compilation : Ndless SDK (nspire-gcc, cross-compilateur ARM926EJ-S)
  • Taille du binaire : ~489 Ko
  • Sources du portage : https://github.com/ztar-dubs/xrick-ndless
